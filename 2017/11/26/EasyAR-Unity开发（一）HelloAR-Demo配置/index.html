EasyAR+Unity开发（一）HelloAR Demo配置

2017-11-26

AR 开发开源 SDK

ARTookit ：https://archive.artoolkit.org/

Vuforia ： https://developer.vuforia.com/downloads/sdk

EasyAR ： http://www.easyar.cn/

####个人感觉：

ARTookit ，相关的使用材料和博客较少，不建议新手入门。

Vuforia ，并未使用过，不过在搜索 AR 相关资料的时候，发现相关的讲解博客较多。

EasyAR ，就是本文所使用的 SDK ，简单易上手。

EasyAR 官网实用功能介绍

支持：官方（简洁版）使用说明

下载： SDK 软件开发工具包的下载地址

问答：提问的场所，一般1天左右都会得到答复

社区：会有官方（详细版）使用说明，和别人的相关 demo 制作

EasyAR 下载以及导入 unity3D

1、用 unity 打开下好的 EasyAR_SDK_2.1.0_Basic_Samples_Unity_2017-09-08 中的 HelloAR 项目

2、打开项目会遇到以下警告，意思是项目的版本过老需要更新，选择update 就行。

3、调整成这样的窗口，Window->layout->2by3

4、在 Project 框中选择 HelloAR->Scenes,打开HelloAR

5、在Hierarchy框中选择 EasyAR_Startup

6、在 EasyAR 官网按照要求创建 APP（示例为Android）

7、将密钥 key 粘贴在5-图红框处

8、unity 发布 Android 的相关配置

参考这篇博客：http://blog.csdn.net/love4399/article/details/77164500

配置好 JDK，SDK后，打开File ->Build Settings,在右侧红色箭头标记地方，填上 包名

9、 使用一台 安卓手机 -> 打开usb调试，用数据线连接电脑，打开 File -> Build and Run，名字随便起，unity 便会在你手机上发布 helloAR 这款 app

最后效果

打开目标图片，用摄像头去扫，出现视频，长方体和球体
