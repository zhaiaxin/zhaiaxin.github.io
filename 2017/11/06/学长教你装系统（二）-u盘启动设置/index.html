学长教你装系统（二） u盘启动设置

2017-11-06

为什么要设置U盘启动？

嗯，good question！

现在装系统的方式有以下3种。

第一种 在电脑上直接运行 setup.exe 程序

优点：安装方法简单；不容易出现开机无引导的问题。

缺点：安装速度慢；只适用于能正常开机的电脑；安装完成后，需要重新下驱动和更新补丁，非常费时间。

过程：解压镜像，运行 setup.exe ，按照提示进行安装。

第二种 使用u盘（原版安装）

优点：安装较简单；没有UEFI和传统模式的限制；基本不会出现错误。

缺点：硬件限制，必须有U盘（容量 8g 及以上）；安装速度慢；安装完成后，需要重新下驱动和更新补丁， 非常费时间。

过程：下面会详细介绍。

第三种 使用u盘（PE系统）

优点：安装快速；下载新版的系统，会省去很多更新补丁的时间；系统安装工具很多，不仅仅是 ghost ，某些 PE 支持 ISO 直接运行；能装好部分驱动 程序

缺点：安装较复杂；硬件和软件限制（需要是事先做好 PE ）；传统和 UEFI 限制，在不同的模式会有不同的问题

过程：之后系列会详细介绍。

如何设置U盘启动？

第一种 设置一键u盘启动快捷键（推荐，但可能电脑 bios 过老没有此热键功能）

win8，win8.1，win10 的用户请留步！（如果不是请直接跳过）

在设置U盘启动之前，务必要关闭电脑的“快速启动”。

后果：如果不关闭，则无法进入boot Setup选择和bios界面！！！

windows 10如何关闭快速启动：http://jingyan.baidu.com/article/ca00d56c7a40e6e99febcf4f.html

知识点普及：

boot menu：

启动菜单，功能为选择从什么地方启动（硬盘HDD，U盘USB，网卡LAN，CD/DVD）。

一键u盘启动快捷键，更加准确地来说，使用这个键可以在不进入bios的情况下，进入boot Setup界面，快速选择启动项。

开机 -> 出现品牌Logo -> 疯狂按一键u盘启动快捷键 -> 进入boot menu界面

小技巧：在出现品牌LOGO之前就可以按，以防电脑速度过快，手速跟不上了。

第二种 设置 bios 中的 boot menu

知识点普及：

bios（Basic Input Output System）：

其实，它是一组固化到计算机内主板上一个ROM芯片上的程序，是计算机运行的第一个软件，其主要功能是为计算机提供最底层的、最直接的硬件设置和控制。

BIOS详解：http://blog.csdn.net/trap94/article/details/50532896
UEFI、BIOS、Secure Boot的关系和知识介绍http://www.cnblogs.com/zengkefu/p/5859954.html
secure boot：http://www.evolife.cn/html/2015/81918.html

五个步骤：

1、secure boot enable 选择disable

2、Boot Mode改成 Legacy Supprot

3、CSM Support 选择 Yes（如果有的话）

4、进入boot menu界面，将USB HDD用+或-（有的是F5或F6）调整至最上面

5、F10保存退出，即可U盘启动
