Hexo+github构建个人博客

2017-10-30

##Hexo 简介

官方说明翻译过来是， Hexo 是一个快捷，简单，强有力的博客框架。它允许你用 markdown 语言（或其他语言）去写博文，而 Hexo 会将这些语言写成的文件（例如：.md）渲染成一个拥有漂亮主题的静态页面。

##Hexo+Github搭建博客

因为关于这方面博客写的非常多，在对比了很多篇讲解，我觉得这位博主写的最为详细易懂。

Hexo+Github搭建博客:http://blog.csdn.net/gdutxiaoxu/article/details/53576018

####注意：
在 Hexo 连接 Github 之前务必配置 SSH (SSH是一种网络协议，用于计算机之间的加密登录)

SSH配置：http://blog.csdn.net/binyao02123202/article/details/20130891

##Hexo 主题选择

Hexo 主题推荐（知乎）：https://www.zhihu.com/question/24422335
Hexo 主题推荐（官网）：https://hexo.io/themes/

##Hexo 主题安装

####1、下载主题
这里以我使用的主题 Vexo （Vexo = = Vue.js Style + Hexo）举例。

1、进入 Vexo 主题的 github 地址：https://github.com/yanm1ng/hexo-theme-vexo

2、复制 git 地址用来克隆

3、按照说明用 gitbash 窗口进行安装

找到放置 Hexo 的文件夹 ，鼠标右键 -选择- git bash ，输入图示内容


4、配置 Hexo 的 _config.yml文件

_config.yml文件配置详解：https://hexo.io/zh-cn/docs/commands.html

5、开始写博客

命令介绍： https://hexo.io/zh-cn/docs/writing.html

##最后
至此，Hexo+github 构建个人博客介绍完毕，关于主题的配置，建议查阅主题的 github 网址的 README.md 文件，一般都会有详尽的说明。

新博客地址：https://zhaiaxin.github.io/
